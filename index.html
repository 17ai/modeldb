ModelDB: a system for managing machine learning models

Download this project as a .zip file Download this project as a tar.gz file

Project Overview

ModelDB is an end-to-end system to manage machine learning models. You can log models, metadata and metrics to ModelDB using our Thrift API or native libraries for spark.ml and scikit-learn. The location for model storage is configurable (e.g. HDFS, local file system) while metadata is stored in a database for easy querying. Data in ModelDB can be queried via the ModelDB frontend or via the Thrift API.

What can you do with ModelDB?

  • Build a central repository for models
  • Make all your models reproducible
  • Manage model versions
  • Compare models across any number of runs
  • Share models, workflows and metrics within and outside your team

How do I use ModelDB?

Start with logging models, metadata and metrics to ModelDB. With minimal changes to your code, you can benefit from ModelDB logging.

Log using APIs

We provide two types of APIs to log information to ModelDB. The light API allows users to log arbitrary (but coarse-grained) information to ModelDB. For example, this API, shown below, can be used to log models, datasets, metrics etc. 

    
        mdb.sync_datasets(datasets)
        mdb.sync_model(model, hyperparameter_dictionary)
        mdb.sync_metrics(model, metric_dictionary)

If you are using spark.ml or scikit-learn, we provide native clients that can collect much more fine-grained information including what operations were used in generating a model. To use these APIs, you replace the relevant functions in your code with the corresponding *_sync functions.

    
        log_reg.fit_sync(X, y)
        one_hot_encoder.transform_sync(X)
        log_reg.predict_sync(X)

Project Status

We are currently testing ModelDB with a small number of companies and research groups. The system will be made open source in early 2017. In the meantime, we are happy to provide early access on a case-by-case basis. Please reach out to mvartak_at_csail.mit.edu.
